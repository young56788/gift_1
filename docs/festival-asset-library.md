# Festival Asset Library

## Library Path
- `public/asset-library/festival`

## What Was Imported

### Pix-Quest
- `public/asset-library/festival/atlases/pix-quest-free`
- `public/asset-library/festival/atlases/pix-quest-full`
- `public/asset-library/festival/references`

Use for:
- plaza ground
- paths
- fences
- tables
- barrels
- small decorative objects
- tile-based scene assembly

Best value for the festival scene:
- strong top-down readability
- clean cozy village palette
- easiest source for building a custom night plaza instead of using a full screenshot

### Pixelwood Valley
- `public/asset-library/festival/buildings`
- `public/asset-library/festival/environment`
- `public/asset-library/festival/props`

Use for:
- houses and shops in the background
- tents and well as town-square anchors
- trees and rocks for framing
- chests and wooden props for side decoration

Best value for the festival scene:
- stronger “cozy valley town” feeling
- better silhouettes for buildings than Pix-Quest
- useful for turning the festival into a real village square instead of a flat map

### Animal Farm
- `public/asset-library/festival/characters/animals`
- `public/asset-library/festival/environment/crops`
- `public/asset-library/festival/environment/ground`

Use for:
- optional animals near the edge of the scene
- crops and farm accents
- tiny environmental filler

Best value for the festival scene:
- low

Do not use as the main festival visual base. It reads more like farm simulation support art than a birthday night event.

## Recommended Use Priority

### Use First
1. `Pix-Quest` atlases for ground, paths, fences, and festival layout
2. `Pixelwood Valley` buildings for background town framing
3. `Pixelwood Valley` trees and wooden props for cozy edge decoration

### Use Carefully
1. `Pixelwood Valley` tents if you want a market-night vibe
2. `Animal Farm` crops only as tiny side detail
3. `Animal Farm` animals only if you want a playful edge detail

### Avoid As Main Scene Pieces
1. `Animal Farm` sprite sheets as core party elements
2. full reference screenshots as final background

## Best Festival Composition From Current Packs
- Background: Pixelwood Valley houses or shops
- Midground: custom plaza built from Pix-Quest tiles
- Side framing: Pixelwood Valley trees, rocks, wooden props
- Foreground props: Pix-Quest objects plus Pixelwood chests or wooden pieces
- Optional accents: Animal Farm crops or one small chicken cluster near the edge
